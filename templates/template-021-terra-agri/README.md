# Terra Agri 永續農業科技網站

## 主題說明
永續農業科技服務網站，呈現農業與科技的完美結合。溫暖的大地色系傳達友善土地與永續發展的理念。

## 設計特色
- **色彩**: 深褐 #3E2723 / 陶土 #8D6E63 / 奶油白 #F5F0E8 / 農業綠 #2E7D32
- **字體**: Bitter（襯線標題）+ Lato（內文）
- **導航**: 固定頂部導航，drop-down 子選單
- **按鈕**: `.terra-btn-earth`（棕色漸層）+ `.terra-btn-green`（綠色邊框）
