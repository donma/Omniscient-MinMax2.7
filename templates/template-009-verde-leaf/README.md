# Verde Leaf — 健康蔬食網站模板

## 設計風格：Organic Flow（自然流動）

Verde Leaf 為健康蔬食、植物性餐廳打造的有機風格網站。核心理念是「自然流動」——用有機曲線、柔和漸層、與大地色系營造一種回到自然懷抱的感覺。

## 特色亮點

- **有機曲線設計**：波浪分隔線貫穿全站，模擬自然的流動感
- **浮動裝飾元素**：背景有緩慢飄動的有機形狀，增加視覺深度
- **薄荷綠配色**：清新自然的色彩搭配森林綠主色調
- **Fraunces + Work Sans**：有機襯線標題配現代清晰內文
- **圓潤卡片**：大圓角、柔和陰影，友善溫暖的視覺體驗

## 色彩系統

```css
--bg: #F0FDF4          /* 薄荷奶綠背景 */
--bg-warm: #FEFCE8     /* 淡蛋黃背景 */
--bg-coral: #FFF1F0    /* 淡珊瑚背景 */
--primary: #15803D     /* 森林綠 */
--accent: #F97316      /* 活力橙 */
--text: #1C1917        /* 暖黑文字 */
--text-light: #78716C  /* 淺棕灰 */
```

## 字體

- **標題**：Fraunces（Google Fonts），有機襯線感
- **內文**：Work Sans（Google Fonts），清晰現代

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊

## 圖片來源

所有圖片來自 [Unsplash](https://unsplash.com)，詳細清單請參閱 `image-sources.md`。

## 響應式設計

- 桌面：完整波浪效果 + 浮動裝飾
- 平板：簡化裝飾，保留曲線
- 手機：全單欄，圓角保留

## JavaScript 功能

- 導航列滾動效果（透明 → 綠色）
- FAQ 折疊互動
- 菜單類別篩選
- 進場淡入動畫（Intersection Observer）
- 手機版導航選單
