# AI Prompt: Apex Finance 峰會金融網站生成

## 網站概述
這是一個專業的金融與保險服務網站，以「雙面板 Portal + 數據儀表板」為設計方向。網站採用深藍色主題，搭配金色點綴，呈現專業、可信賴的金融機構形象。

## 設計方向
- **風格**: 商務專業、數據驅動、儀表板式管理介面
- **色彩**: 深海軍藍 #0B1D3A、金色 #D4A843、白色 #F0F4FA
- **字體**: Playfair Display（標題）+ Inter（內文）
- **佈局**: 左右雙面板，左側固定導航 + 右側內容區

## 目標受眾
- 高淨值個人客戶
- 企業主
- 尋求專業理財服務的投資人
- 需要保險規劃的家庭

## 功能需求

### 頁面結構
1. **首頁** - 登入後的儀表板頁面，包含：
   - 歡迎訊息與組合總覽
   - 4個統計數據卡片（大數字展示）
   - 4個金融產品卡片
   - 3個最新市場洞察文章

2. **關於我們** - 公司介紹、里程碑、核心價值

3. **服務項目** - 金融服務項目展示、服務流程說明

4. **金融產品** - 產品列表（篩選功能）、熱門產品表格、保險產品專區

5. **產品詳情** - 單一產品詳細資訊、特色說明、利率比較

6. **計算機** - 存款利息試算、房貸月付額試算

7. **市場洞察** - 文章列表（分類篩選）、精選文章

8. **文章詳情** - 完整文章內容、相關文章推薦

9. **專業團隊** - 經營團隊、理財顧問、專業資格認證

10. **聯絡我們** - 聯絡表單、據點資訊、服務時間

11. **FAQ** - 常見問題（手風琴樣式）、分類展示

12. **人才招募** - 人才福利、職缺列表、履歷投遞

## 內容要求

### 金融產品資料
- 優利活存帳戶：利率 2.85%，最低 100 萬
- 全球成長基金：年報酬 12.8%
- 樂活退休基金：年報酬 6.5%
- 房貸專案：利率 1.6% 起

### 數據展示
- 總資產：2,847 萬
- 投資收益：364 萬
- 保障額度：1,200 萬
- 年化報酬率：12.8%

## 技術要求
- 使用 Google Fonts（Playfair Display + Inter）
- FontAwesome 圖示庫
- 回應式設計（支援桌面與平板）
- 純 HTML/CSS/JavaScript
- 無需後端

## 圖片需求
使用 Unsplash 高品質金融相關圖片：
- 總部大樓/辦公環境
- 金融圖表/數據視覺化
- 商務人士/團隊照片
- 城市天際線
- 會議/洽谈场景

## 輸出格式
- 12 個完整 HTML 頁面
- 統一的 CSS 樣式表（style.css）
- JavaScript 交互（選單、篩選、試算）
- 所有中文字串完整

## 設計細節
- 左侧边栏宽度：260px，固定定位
- 卡片圆角：12px
- 按钮圆角：8px
- 间距系统：8px 为基础单位
- 金色强调线：4px 顶部边框
