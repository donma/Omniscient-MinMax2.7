# LUMIÈRE Photo — Design Brief

## 1. Concept & Vision

LUMIÈRE 為婚禮攝影與精品相簿打造的 Full-screen Gallery 網站。核心理念是「每一刻都值得被溫柔以待」——以極致乾淨的版面，讓照片自己說故事。

## 2. Design Language

- **Palette**: 奶油白 #FAF8F5 / 啞光金 #C9A84C / 深炭 #2A2A2A
- **Fonts**: Cormorant Garamond (標題，優雅襯線) + Jost (內文，現代幾何)
- **Aesthetic**: 婚禮感的溫柔奶油色調，金色點綴精緻感

## 3. Layout & Structure

- Full-screen Hero 婚禮照
- Masonry Grid 作品集展示
- 雙欄佈局：圖 / 文交替
- 精緻的字距與留白

## 4. Pages

index, about, portfolio, portfolio-detail, pricing, testimonials, journal, journal-detail, FAQ, service, contact, booking

## 5. Technical

Pure HTML/CSS/JS, Google Fonts, Unsplash images, CSS grid masonry layout
