# Prestige Realty — 豪宅不動產網站模板

## 設計風格：左右雙側欄 + Sticky Filter Panel

Prestige Realty 為高端豪宅不動產打造的 Portal 風格網站。核心理念是「尊貴體驗，從第一眼開始」——用 Editorial 雜誌風格，呈現頂級豪宅的非凡氣質。

## 特色亮點

- **左右雙側欄導航**：Portal 風格，資訊豐富而不雜亂
- **Sticky Filter Panel**：固定篩選面板，快速找到理想物件
- **象牙白配色**：#FFFFF0 / 啞光金 #B8860B / #333333
- **Playfair Display + Montserrat**：優雅襯線標題，现代无衬线正文
- **Editorial 雜誌風格**：大圖主導，金色點綴尊貴感

## 色彩系統

```css
--bg: #FFFFF0      /* 象牙白 */
--gold: #B8860B     /* 啞光金 */
--text: #333333     /* 深灰文字 */
--text-light: #777777 /* 淺灰文字 */
--white: #FFFFFF    /* 純白 */
--border: #E5E0D0   /* 米白邊框 */
```

## 字體

- **標題**：Playfair Display（Google Fonts），優雅襯線
- **內文**：Montserrat（Google Fonts），現代幾何無襯線

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊

## 圖片來源

所有圖片來自 [Unsplash](https://unsplash.com)，詳細清單請參閱 `image-sources.md`。

## 響應式設計

- 桌面：雙側欄導航與 Sticky 篩選面板
- 平板：簡化側欄，篩選面板折疊
- 手機：全單欄，側欄改為底部導航

## JavaScript 功能

- Sticky 篩選面板聯動
- 物件卡片 Hover 效果
- 圖片燈箱展示
- 手機版側欄導航選單
- 滾動淡入動畫
