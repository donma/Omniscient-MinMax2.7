# AI Prompt: Terra Agri 永續農業科技網站生成

## 網站概述
這是一個永續農業科技服務網站，以「溫暖的大地色系 + 現代農業科技」為設計方向。網站呈現農業與科技的完美結合，传达友善土地、永續發展的理念。

## 設計方向
- **風格**: Warm Earthy、科技農業、回應式引導式介面
- **色彩**: 深褐 #3E2723、陶土 #8D6E63、奶油白 #F5F0E8、農業綠 #2E7D32、淺綠 #4CAF50
- **字體**: Bitter（標題，襯線字）+ Lato（內文）
- **佈局**: 固定頂部導航 + 下拉選單 + 全寬 Hero + 資訊卡片

## 目標受眾
- 小型到大型農戶
- 農業企業與合作社
- 對永續農業感興趣的消費者

## 功能需求

### 頁面結構
1. **首頁** - 全寬 Hero、品牌標語、解決方案總覽、成功案例、數據展示
2. **關於我們** - 公司簡介、使命願景、發展歷程、的合作夥伴
3. **解決方案** - 6大農業科技解決方案（智慧灌溉、土壤檢測、無人機、AI病蟲害、產銷管理、氣象預警）
4. **解決方案詳情** - 單一解決方案詳細介紹、功能特色、操作介面說明
5. **市場數據** - 農產品價格走勢、即時產地行情、歷史數據圖表
6. **農產品** - 友善耕種農產品展示、產銷履歷、購買連結
7. **知識庫** - 農業知識文章、種植技巧、友善環境文章列表
8. **知識庫詳情** - 文章內容、相關文章推薦
9. **團隊成員** - 農業專家與科技研發團隊介紹
10. **影響力** - 累積服務面積、作物產量提升、節水數據、農戶數
11. **聯絡我們** - 顧問表單（姓名、Email、農場面積感興趣方案）、據點資訊
12. **FAQ** - 常見問題（折疊手風琴式）、分類標籤

## 技術要求
- 使用 Google Fonts（Bitter + Lato）
- FontAwesome 圖示庫
- SVG 圖示（Leaf/Agriculture theme）
- 回應式設計
- 純 HTML/CSS/JavaScript

## 圖片需求
使用 Unsplash 高品質農業相關圖片：
- 智慧灌溉系統
- 農業無人機
- 有機農田全景
- 土壤與作物特寫
- 農業科技數據視覺化

## 輸出格式
- 12 個完整 HTML 頁面
- 統一的 CSS 樣式表（style.css）
- JavaScript 交互（下拉選單、FAQ 展開、過濾器）
- 所有中文字串完整

## 設計細節
- 導航列：高度 80px，固定頂部，drop-shadow
- 按鈕：.terra-btn-earth（棕色漸層）+ .terra-btn-green（綠色邊框）
- 卡片：.terra-solution-card（圓角 16px，陰影）
- 間距系統：40px / 60px / 80px 三層
