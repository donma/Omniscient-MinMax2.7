# Humming Tech 蜂鳥科技 — AI Prompt

使用以下 prompt 生成類似的科技商務網站：

---

## Prompt

```
設計一個名為「蜂鳥科技 Humming Tech」的科技公司網站。

## 視覺風格
- Dark Bento Grid 暗金美學
- 主色：啞光黑 #0c0c0c / 金色 #d4a843 / 深黑 #1a1a1a
- 背景：純黑，內容區塊深灰
- 襯線裝飾：啞光金色細線

## 字體
- 標題：DM Sans（粗體數字顯示）
- 內文：Plus Jakarta Sans

## 首頁區塊
1. 全螢幕 Hero：巨大粗體數字（可用計數動畫）+ 公司標語
2. Bento Grid 特點展示：4-6 個不規則方格， hover 放大效果
3. 服務項目：4 個卡片， icon + 標題 + 描述
4. 數據展示：豎排大數字 + 標籤
5. 客户 LOGO 展示（橫向滾動）
6. CTA 區塊：金色漸層按鈕

## 內頁（12頁）
- about.html：品牌故事 + 團隊成員網格
- services.html：4 項核心服務詳細
- service-detail.html：單項服務詳細
- portfolio.html：專案子網格展示
- pricing.html：三方案定價對比表
- testimonials.html：客户成功案例引用
- blog.html：科技趨勢文章列表
- blog-detail.html：文章內頁
- faq.html：常見技術問題
- contact.html：聯絡表單 + 據點
- careers.html：人才招募

## 設計要求
- 繁體中文
- Unsplash 科技風格圖片
- 滑鼠 hover 有金色光暈效果
- 全域滾動流暢
```

---

## 使用說明

此 prompt 可用於：
- ChatGPT / Claude 等 AI 助手生成完整網站
- Figma / Sketch 設計參考
- Midjourney 生成 UI 視覺概念

建議 AI 輔助次數：3-5 次迭代，根據輸出結果持續優化描述。
