# Spark Academy 教育培訓網站模板

## 模板介紹

Spark Academy 是一套完整的教育培訓網站模板，採用現代卡片方格設計與創新的進度追蹤系統。專為線上學習平台、訓練機構、補習班、學校等教育相關網站設計。

## 主要特色

### 設計亮點
- **明亮黃色 (#F7C948)** 與 **深藍色 (#1E3A5F)** 的專業配色
- **Poppins** 字體用於標題，**Inter** 字體用於內文
- 卡片式佈局，視覺清晰層次分明
- 前所未有的學習進度追蹤系統
- 完全響應式設計，支援手機、平板、桌面

### 完整頁面
本模板包含 12 個完整頁面：
1. **首頁 (index.html)** - Hero區塊、進度追蹤、熱門課程、統計數據、導師介紹、定價方案
2. **關於我們 (about.html)** - 公司介紹、使命願景、核心價值、發展歷程、團隊成員
3. **課程總覽 (courses.html)** - 所有課程展示
4. **課程詳情 (course-detail.html)** - 單一課程詳細頁面
5. **導師團隊 (mentors.html)** - 導師專業背景與經驗
6. **訂閱方案 (pricing.html)** - 三種定價方案與比較表
7. **學員推薦 (testimonials.html)** - 成功案例與推薦語
8. **常見問題 (faq.html)** - FAQ 手風琴問答
9. **部落格 (blog.html)** - 文章列表
10. **部落格文章 (blog-detail.html)** - 單一文章頁面
11. **聯絡我們 (contact.html)** - 聯絡表單與資訊
12. **人才招募 (careers.html)** - 職缺列表與福利

## 使用方式

1. **下載模板**：直接下載整個資料夾
2. **修改內容**：根據需求修改文字、圖片、連結
3. **自訂樣式**：修改 `assets/css/style.css` 中的 CSS 變數
4. **上線部署**：上傳至你的網頁伺服器
