# EduFlow Template

線上學習平台Metro Tiles風格網站，靛藍主色搭配翡翠綠點綴，現代教育科技美學。

## 特色

- **Metro Tiles 佈局**：Windows 8風格方塊，大小混合排列
- **左右導航結構**：Dashboard風格，左側圖標導航
- **進度視覺化**：圓環進度、線性進度條、完成百分比
- **Plus Jakarta Sans**：現代幾何感字體
- **冷灰白底**：專業清爽的教育平台調性
