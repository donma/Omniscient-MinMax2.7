# Aurum Jewelry — Design Brief

## 1. Concept & Vision

Aurum Jewelry 為奢華珠寶與婚禮精品打造的高貴網站。核心理念是「永恆的價值」——以極致黑色的背景襯托玫瑰金與象牙白的珠寶精品，創造高級珠寶雜誌的質感。

## 2. Design Language

- **Palette**: 極致黑 #0A0A0A / 玫瑰金 #B76E79 / 象牙 #FAF9F6 / 深炭 #1C1C1E
- **Fonts**: Cinzel (標題，優雅羅馬大寫) + Montserrat (內文)
- **Aesthetic**: 奢華編輯風，大量留黑，讓珠寶自己發光

## 3. Layout & Structure

- Editorial Luxury：非對稱編排
- 大圖珠寶展示
- 金屬字體標題
- 滑動式內容切換

## 4. Pages

index, about, collections, collection-detail, engagement, wedding, bespoke, press, team, contact, appointment, faq

## 5. Technical

Pure HTML/CSS/JS, Google Fonts, Unsplash images, CSS editorial grid
