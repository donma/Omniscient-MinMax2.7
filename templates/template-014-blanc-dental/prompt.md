# Blanc Dental — Design Brief

## 1. Concept & Vision

Blanc Dental 為現代牙科診所打造的超乾淨網站。核心理念是「微笑是您最好的配件」——以極致簡潔、專業的視覺，讓患者感到安心與信任。薄荷綠與純白的搭配傳遞清新、專業、值得信賴的感受。

## 2. Design Language

- **Palette**: 純白 #FFFFFF / 薄荷綠 #E8F5F0 / 深灰 #333333 / 柔和灰 #F5F5F5
- **Fonts**: Nunito Sans (標題 + 內文，現代圓潤)
- **Aesthetic**: 極度乾淨、極度專業，讓患者第一眼就感到安心

## 3. Layout & Structure

- Clean Minimal 設計，大量留白
- Card Grid 服務項目展示
- 雙欄佈局：圖 / 文交替
- 清晰的資訊階層

## 4. Pages

index, about, services, doctors, appointments, testimonials, blog, blog-detail, contact, insurance, faq, gallery

## 5. Technical

Pure HTML/CSS/JS, Google Fonts, Unsplash images, CSS clean grid
