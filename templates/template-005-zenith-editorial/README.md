# Zenith Editorial Template

專業服務文字導向編輯風格網站樣板，黑白金配色，極致留白，襯線字體建立權威形象。

## 特色

- **編輯風格排版**：非對稱網格，大量留白，襯線字體
- **黑白金配色**：啞光金點綴，極致專業感
- **滾動動畫**：Intersection Observer 驅動的淡入效果
- **FAQ 折疊**：純 CSS + JS 實現的展開/收起
- **響應式設計**：完整適配桌面、平板、手機
