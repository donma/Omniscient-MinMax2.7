# Prestige Realty — Design Brief

## 1. Concept & Vision

Prestige Realty 為高端不動產與豪宅經紀打造的 Editorial 網站。核心理念是「不凡的選擇需要不凡的呈現」——以頂級媒體雜誌的版面，演繹豪宅的稀有與尊貴。

## 2. Design Language

- **Palette**: 象牙白 #FFFFF0 / 啞光金 #B8860B / 深灰 #333333
- **Fonts**: Playfair Display (標題，經典襯線) + Montserrat (內文，現代簡約)
- **Aesthetic**: 豪宅雜誌編輯風，大量留白，讓房產本身說話

## 3. Layout & Structure

- Magazine Editorial：大圖主導 + 文字疊加
- 雙欄編輯排版
- 精選豪宅幻燈片展示
- 統計數據動畫（已售/在售/總價值）

## 4. Pages

index, about, properties, property-detail, services, market-insights, team, testimonials, contact, mortgage, careers, faq

## 5. Technical

Pure HTML/CSS/JS, Google Fonts, Unsplash images, CSS editorial grid
