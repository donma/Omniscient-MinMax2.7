# Apex Finance 峰會金融網站模板

## 模板介紹

Apex Finance 峰會金融是一款專為金融機構設計的專業網站模板，採用雙面板 Portal 佈局與數據儀表板風格，呈現高端、可信賴的金融服務形象。

## 設計特色

### 色彩系統
- **深海軍藍** `#0B1D3A` - 主色調，傳達專業與穩重
- **璀璨金** `#D4A843` - 強調色，用於重點資訊與按鈕
- **雲白色** `#F0F4FA` - 背景色，營造明亮的視覺效果

### 字體搭配
- **Playfair Display** - 標題字體，優雅的襯線體
- **Inter** - 內文字體，清晰易讀的無襯線體

### 佈局結構
- 左側固定導航欄（260px）
- 右側內容區自適應
- 響應式設計支援平板裝置

## 組件特色

### 統計數據卡片
- 金色頂部邊線裝飾
- 大數字展示（48px Playfair Display）
- 變化百分比即時更新

### 金融產品卡片
- 深色背景突出內容
- 利率/報酬率金色標示
- 滑鼠懸停陰影效果

### 手風琴 FAQ
- 點擊展開/收合
- 平滑過渡動畫
- 金色箭頭指示

### 篩選面板
- 下拉式選單
- 即時篩選結果
- 清除重置功能

## 自訂項目

如需修改品牌色彩或內容，可編輯 `assets/css/style.css` 中的 CSS 變數：

```css
:root {
    --navy: #0B1D3A;
    --gold: #D4A843;
    --white: #F0F4FA;
}
```
