# Brasserie 1888 — Design Brief

## 1. Concept & Vision

Brasserie 1888 為精釀啤酒與手工餐酒打造的 Split Image + Text 網站。核心理念是「手工的驕傲」——每一杯啤酒、每一道菜都有它背後的故事。用溫暖的深琥珀色調與銅色金屬點綴，營造精釀啤酒廠的質感氛圍。

## 2. Design Language

- **Palette**: 深琥珀 #2C1810 / 銅色 #B87333 / 奶油白 #F5FDE8 / 深炭 #1A1A1A
- **Fonts**: Playfair Display (標題，經典襯線) + Source Sans 3 (內文)
- **Aesthetic**: 手工精釀的溫暖與精準，復古酒吧的氛圍

## 3. Layout & Structure

- Split Image + Text 交替排版
- 大圖啤酒/食物展示
- 菜單分組展示
- 活動與事件公告

## 4. Pages

index, about, menu, events, gallery, story, reservation, taproom, contact, careers, faq, wholesale

## 5. Technical

Pure HTML/CSS/JS, Google Fonts, Unsplash images
