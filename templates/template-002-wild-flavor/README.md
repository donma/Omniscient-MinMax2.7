# 野味餐酒｜使用說明

## 樣板介紹
這是一套專為 Fine Dining 餐酒館設計的網站樣板，採用 Magazine Editorial 風格，適合強調文化底蘊與食材故事的當代餐廳品牌。

## 適合產業
Fine Dining、餐酒館、預約制私廚、咖啡甜點、調酒吧

## 版型說明
- **主版型**：Magazine Editorial（編輯式非對稱網格）
- **字體**：Playfair Display（標題）＋ Noto Sans TC（內文）
- **色彩**：大地色系（主色 `#8b4513`、重點 `#4a7c59`）

## 如何修改品牌名稱
搜尋「野味餐酒」與「Wild Flavor」，替換為你的品牌名稱。

## 如何替換圖片
所有圖片來自 Unsplash。將 `src="https://images.unsplash.com/..."` 替換為 `src="assets/img/your-image.jpg"`。
