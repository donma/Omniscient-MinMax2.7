# NOIR Atelier — 當代時尚服飾網站

## 主題說明
NOIR Atelier 為當代時尚服飾品牌網站，設計方向為「編輯式奢華」。全站以極致黑背景襯托米白與金色，創造高端時尚雜誌質感。

## 設計特色
- **色彩**: 極致黑 #0D0D0D / 灰黑 #1A1A1A / 暖白 #F5F0EB / 金色 #C9A96E
- **字體**: Cormorant Garamond（編輯感強烈襯線）+ Libre Franklin
- **首頁**: 無傳統導航列，全屏 Hero 豎向滾動切換編輯式板塊
- **佈局**: Editorial Grid 非對稱欄位（col-5 / col-7）
