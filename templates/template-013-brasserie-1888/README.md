# Brasserie 1888 — 精釀啤酒網站模板

## 設計風格：Horizontal Scroll 全屏橫向滑動

Brasserie 1888 為精釀啤酒與手工餐酒打造的 Horizontal Scroll 全屏橫向滑動網站。核心理念是「釀造工藝的極致呈現」——用深琥珀底配銅色，營造溫暖的啤酒館氛圍。

## 特色亮點

- **Horizontal Scroll 全屏滑動**：左右滑動切換面板，沈浸式體驗
- **左側固定導航**：始終可見的導航選單
- **深琥珀配色**：#2C1810 / 銅色 #B87333 / #F5FDE8
- **滑動面板內容**：每個面板一個主題，視覺冲击力强
- **啤酒館氛圍**：溫暖、質感、專業

## 色彩系統

```css
--bg: #2C1810      /* 深琥珀 */
--copper: #B87333  /* 銅色 */
--cream: #F5FDE8   /* 奶油綠 */
--text: #F5FDE8    /* 淺色文字 */
--dark: #1A0F0A    /* 深咖啡 */
```

## 字體

- **標題**：Playfair Display（Google Fonts），復古優雅襯線
- **內文**：Inter（Google Fonts），現代簡潔

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊

## 圖片來源

所有圖片來自 [Unsplash](https://unsplash.com)，詳細清單請參閱 `image-sources.md`。

- 桌面：完整 Horizontal Scroll 效果
