# Blanc Dental — 牙科診所網站模板

## 設計風格：弧形波浪分隔線 + 步驟向導

Blanc Dental 為現代牙科診所打造的專業網站。核心理念是「專業、清新、可信賴」——用薄荷綠與純白配色，搭配弧形波浪分隔線，創造一個讓人安心的醫療體驗。

## 特色亮點

- **弧形波浪分隔線**：柔和的視覺過渡，友善就醫體驗
- **步驟向導設計**：就診流程視覺化，降低焦慮感
- **薄荷綠配色**：#FFFFFF / #E8F5F0 / #0D9488
- **重疊圓形圖片**：獨特的圖文排版，時尚且專業
- **極度專業清新**：醫療感的可信賴形象

## 色彩系統

```css
--bg: #FFFFFF      /* 純白 */
--mint: #E8F5F0    /* 薄荷綠背景 */
--primary: #0D9488 /* 主色青綠 */
--text: #1F2937    /* 深灰文字 */
--muted: #6B7280   /* 淺灰文字 */
```

## 字體

- **標題**：Noto Sans TC（Google Fonts），繁體中文字體
- **內文**：Noto Sans TC（Google Fonts），現代清晰

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊
