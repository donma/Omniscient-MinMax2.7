# Template 024: Heartbeat NGO Website Generation Prompt

## 概述
生成一個暖色調的非營利組織網站，名為「心跳基金會」，主打故事牆式內容呈現，傳遞溫暖與社會影響力。

## 品牌設定
- **基金會名稱**: 心跳基金會 (Heartbeat Foundation)
- **口號**: 用愛守護每一個生命
- **成立年份**: 2010年
- **總部**: 台北市大安區和平東路二段100號
- **電話**: (02)2700-7890
- **信箱**: love@heartbeat.org.tw

## 色彩系統
- **Primary Orange**: #E8572A (溫暖橙)
- **Soft Cream**: #FFF3E0 (奶油色背景)
- **Dark Brown**: #2D2D2D (深棕文字)
- **Warm White**: #FFFAF5 (暖白底色)
- **Accent Teal**: #00897B (點綴青色)
- **Light Orange**: #FFAB91 (淺橙)
- **Medium Brown**: #5D4E37 (中棕)

## 字體設定
- **標題字體**: Merriweather (襯線字體，溫暖感)
- **內文字體**: Open Sans (無襯線，易讀性)

## 設計方向
1. **暖色影響**: 以橙色為主色，傳遞溫暖與希望
2. **故事牆**: 交替式左右版面，大圖配文字，強調情感連結
3. **Impact First**: 數據統計突出顯示，用數字說故事
4. **情感號召**: 志願服務與捐贈呼籲明顯可見

## 頁面結構 (12頁)

### 1. index.html (首頁)
- 大型Hero區塊：情緒化照片背景、深色覆蓋層、呼籲行動按鈕
- 影響力數據計數器 (4個大數字)
- 故事牆區塊 (2-3個交替故事)
- 服務項目卡片展示 (3個計畫)
- 引言橫幅
- 最新消息卡片 (3則)
- 志工報名表單
- 頁尾

### 2. about.html (關於我們)
- 頁面標題區
- 組織簡介
- 發展歷程時間軸
- 核心理念價值
- 里程碑成果

### 3. mission.html (使命與願景)
- 使命宣言
- 願景描述
- 核心價值 (4-6項)
- 策略目標

### 4. programs.html (服務項目總覽)
- 頁面標題區
- 6-8個計畫卡片網格
- 每個計畫含圖片、標題、簡介

### 5. program-detail.html (計畫詳細頁)
- 麵包屑導航
- 計畫大圖橫幅
- 計畫介紹
- 目標與策略
- 過往成果
- 相關報導連結

### 6. impact.html (影響力成果)
- 頁面標題區
- 大型數據統計展示
- 年度報告摘要
- 個案故事分享
- 合作夥伴標誌

### 7. team.html (團隊介紹)
- 頁面標題區
- 執行團隊 (4-6人)
- 董事會成員
- 專業顧問

### 8. news.html (最新消息列表)
- 頁面標題區
- 消息分類篩選
- 新聞卡片網格 (6-9則)
- 分頁導航

### 9. news-detail.html (消息詳情)
- 麵包屑導航
- 文章大圖
- 文章標題與日期
- 文章內容
- 相關文章推薦

### 10. donate.html (捐贈頁面)
- 頁面標題區
- 捐贈方案選擇 (3種金額)
- 自訂金額輸入
- 線上捐贈表單
- 離線捐贈資訊
- 發票說明
- 公益捐贈抵稅說明

### 11. volunteer.html (志工招募)
- 頁面標題區
- 志工故事分享
- 服務項目說明
- 志工報名表單
- 常見問題

### 12. contact.html (聯絡我們)
- 頁面標題區
- 聯絡資訊卡片
- 聯絡表單
- 地圖嵌入區

- 手機導航選單切換

## 圖片來源
所有圖片使用 Unsplash，格式:
- Hero 圖: ?w=1200&q=85
- 內容圖: ?w=800&q=85
- 團隊照: ?w=400&q=85

建議分類:
- 兒童教育: community, children, education
- 志願服務: volunteer, helping, charity
- 社會影響: social impact, support, care
- 團隊合作: team, professionals

## 技術規範
- 單頁 HTML 模板
- 純 CSS (不使用 Tailwind)
- Vanilla JavaScript
- Font Awesome 6.x CDN
- Google Fonts CDN

## 生成要求
1. 所有檔案使用 UTF-8 編碼
2. 中文內容使用 zh-TW lang 屬性
3. 所有圖片使用有效 Unsplash URL
4. 響應式斷點: 1024px, 768px, 480px
5. 保持一致的品牌識別
6. 完整可運作的表單 (外觀)
