# Aurum Jewelry — 奢華珠寶網站模板

## 設計風格：無 Nav 巨型 Editorial + Drop-cap

Aurum Jewelry 為奢華珠寶打造的無導航列超大字首頁網站。核心理念是「極致奢華，編輯呈現」——用極致黑底配啞光金，Drop-cap 首字與多欄編輯式內容，創造雜誌般的視覺震撼。

## 特色亮點

- **無 Nav 超大字首頁**：打破常規，第一眼就驚艷
- **Drop-cap 首字**：編輯式排版，雜誌風格
- **多欄編輯內容**：豐富的視覺層次
- **極致黑配色**：#0A0A0A / 玫瑰金 #B76E79 / #FAF9F6
- **啞光金點綴**：精緻奢華感

## 色彩系統

```css
--bg: #0A0A0A     /* 極致黑 */
--gold: #B76E79   /* 玫瑰金 */
--cream: #FAF9F6  /* 奶油白文字 */
--white: #FFFFFF  /* 純白 */
--border: #2A2A2A /* 深灰邊框 */
```

## 字體

- **標題**：Cormorant Garamond（Google Fonts），奢華襯線
- **內文**：Raleway（Google Fonts），優雅幾何

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊
