# LUMIÈRE Photo — 婚禮攝影網站模板

## 設計風格：Full-screen Gallery + Masonry Grid

LUMIÈRE 為婚禮攝影與精品相簿打造的 Full-screen Gallery 網站。核心理念是「每一刻都值得被溫柔以待」——以極致乾淨的版面，讓照片自己說故事。

## 特色亮點

- **Full-screen Gallery**：全螢幕婚禮照片展示
- **Masonry Grid**：瀑布流作品集，視覺豐富
- **奶油白配色**：#FAF8F5 / 啞光金 #C9A84C / 深炭 #2A2A2A
- **Cormorant Garamond + Jost**：優雅襯線標題，現代幾何內文
- **婚禮感美學**：溫柔色調，金色點綴精緻感

## 頁面清單（12 頁）

| 頁面 | 檔案 | 說明 |
|------|------|------|
| 首頁 | `index.html` | Full-screen Hero、Masonry 作品集預覽 |

| 作品詳情 | `portfolio-detail.html` | 單一婚禮作品詳細頁面 |
| 方案與報價 | `pricing.html` | 婚禮拍攝方案與收費 |
| 客戶推薦 | `testimonials.html` | 新人推薦與評價 |
| 日常記錄 | `journal.html` | 攝影日誌與花絮 |
| 文章詳情 | `journal-detail.html` | 日記文章內頁 |
| 常見問題 | `faq.html` | 婚禮拍攝 FAQ |
| 服務項目 | `service.html` | 婚禮、訂婚、寫真等服務 |
| 聯絡我們 | `contact.html` | 聯絡表單與諮詢 |
| 預約檔期 | `booking.html` | 檔期預約表單 |

## 色彩系統

```css
--bg: #FAF8F5      /* 奶油白 */
--gold: #C9A84C    /* 啞光金 */
--text: #2A2A2A    /* 深炭 */
--white: #FFFFFF   /* 純白 */
--border: #E8E4DE  /* 米白邊框 */
```

## 字體

- **標題**：Cormorant Garamond（Google Fonts），優雅襯線
- **內文**：Jost（Google Fonts），現代幾何無襯線

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊

## 圖片來源

所有圖片來自 [Unsplash](https://unsplash.com)，詳細清單請參閱 `image-sources.md`。

## 響應式設計

- 桌面：Full-screen Gallery 與 Masonry Grid 效果
- 平板：簡化網格，保留大圖展示
- 手機：全單欄，圖片垂直堆疊

## JavaScript 功能

- Full-screen 圖片切換滑動
- Masonry Grid 瀑布流佈局
- 圖片燈箱放大效果
- 手機版導航選單
- 滾動淡入動畫
