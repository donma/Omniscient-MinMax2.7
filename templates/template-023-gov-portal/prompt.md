# GovPortal 政府入口網站 — AI Prompt

使用以下 prompt 生成類似的政府機構入口網站：

---

## Prompt

```
設計一個名為「GovPortal」的政府機構入口網站。

## 視覺風格
- Official Clean 正式清晰 + 網格導航
- 主色：官方深藍 #1A3A6B / 淺藍 #2B5BAE / 白底
- 背景：白色與淺灰 #F5F7FA
- 字體：Noto Sans TC（Google Fonts 正體中文）

## 版型特色
1. 固定頂部導航，含機關標誌、服務連結、搜尋框、登入按鈕
2. 麵包屑導航（首頁 > 服務資訊 > [當前頁面]）
3. 服務專區：6 欄格狀服務 tiles， hover 藍色高亮
4. 公告區：日期標籤 + 標題 + 連結箭頭
5. 數據儀表板：關鍵政績數字
6. 表單區：清除的政府風格表單（標籤在上，輸入框在下手機版響應式）

## 內頁（12頁）
- index.html：首頁，服務總覽、公告、政策專區
- services.html：服務一覽
- announcements.html：公告列表
- announcement-detail.html：公告詳情
- forms.html：表單下載
- faq.html：常見問題
- links.html：相關連結
- about.html：關於我們
- contact.html：聯絡我們
- portal.html：民眾服務入口（登入）
- news.html：最新消息列表
- news-detail.html：消息詳情

## 設計要求
- 繁體中文
- Unsplash 政府/建築風格圖片
- 高對比、清晰易讀
- WCAG 2.1 AA 對比度
```

---

## 使用說明

此 prompt 可用於：
- ChatGPT / Claude 等 AI 助手生成完整網站
- Figma / Sketch 設計參考
- Midjourney 生成 UI 視覺概念

建議 AI 輔助次數：3-5 次迭代。
