# Template 022 - NEON MATRIX (霓虹暗色 + 遊戲產業)

## Template Overview
This is a cyberpunk-inspired gaming website template with a dark theme and neon accents. It features dynamic grid layouts, glow effects, and a futuristic gaming aesthetic.

## Design Direction
- **Color Palette**: Deep Black #0D0D0D, Neon Green #00FF9F, Neon Pink #FF0080, Purple #9D00FF, Dark Gray #1A1A1A
- **Typography**: Orbitron (headings, futuristic feel) + Rajdhani (body text)
- **Visual Style**: Cyberpunk gaming aesthetic, neon glow effects, animated grid backgrounds

## CSS Structure (500+ lines)
All CSS classes use the `-neon` suffix for uniqueness:
- `.header-neon` - Fixed header with Orbitron logo
- `.hero-neon` - Full-screen hero with animated grid
- `.btn-neon-green`, `.btn-neon-pink` - Neon-styled buttons
- `.game-card-neon` - Game card components
- `.team-card-neon` - Esports team cards
- `.news-card-neon` - News article cards
- `.footer-neon` - Full footer section

## Pages Included (12 total)
1. `index.html` - Homepage with hero, featured games, stats, news, esports teams
2. `games.html` - Game library with filters and grid layout
3. `game-detail.html` - Individual game page with details and reviews
4. `news.html` - News listing page
5. `news-detail.html` - News article detail page
6. `esports.html` - Esports section with teams, events, streamers
7. `community.html` - Community forum/discussion area
8. `merch.html` - Merchandise store
9. `about.html` - About us page with company history
10. `contact.html` - Contact form and information
11. `faq.html` - FAQ with expandable answers
12. `careers.html` - Job listings and application form

## JavaScript Features
- Mobile navigation toggle
- Smooth scroll navigation
- FAQ accordion functionality
- Animated stat bars on scroll
- Card animations on scroll
- Glitch text effect on hover
- Form validation
- Counter animations
- Header scroll effects

## Image Sources
All images sourced from Unsplash with appropriate query parameters:
- Game images: gaming-related photography
- Team logos: esports/gaming themed images
- Hero backgrounds: cyberpunk/futuristic imagery

## How to Use
1. Copy all files to your project directory
2. Update links in navigation to match your file structure
3. Customize content (games, news, team info) as needed
4. Replace placeholder images with your own assets
5. Update contact information in footer and contact page

## Responsive Breakpoints
- Desktop: 1200px+ (full grid layouts)
- Tablet: 992px-1199px (reduced columns)
- Mobile: <992px (single column, hamburger menu)

## Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)
- CSS Grid and Flexbox for layouts
- CSS animations and transitions
- Backdrop-filter for glass effects
