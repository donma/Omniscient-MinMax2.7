# Humming Tech 蜂鳥科技 — 科技商務網站模板

## 設計風格：Bento Grid 暗金美學

Humming Tech 為科技公司打造的 Bento Grid 暗金美學網站。核心理念是「精準而有力」——用暗金色調與卡片式佈局，創造一個具未來感的專業形象。

## 特色亮點

- **Bento Grid 卡片式佈局**：不規則方格交錯排版，層次分明
- **暗金配色**：啞光黑 #0c0c0c / 金色 #d4a843 / 深黑 #1a1a1a
- **DM Sans + Plus Jakarta Sans**：現代幾何無襯線，字體銳利精準
- **大數字展示**：計數動畫效果，視覺衝擊力強
- **卡片 Hover 效果**：金色光暈放大效果

## 色彩系統

```css
--bg: #0c0c0c      /* 啞光黑 */
--surface: #1a1a1a /* 深黑 */
--gold: #d4a843    /* 暗金 */
--text: #ffffff    /* 白色文字 */
--muted: #888888   /* 灰階文字 */
```

## 字體

- **標題**：DM Sans（Google Fonts），幾何粗體數字顯示
- **內文**：Plus Jakarta Sans（Google Fonts），現代精準

## 安裝與使用

1. 複製整個資料夾至您的專案
2. 直接在瀏覽器中開啟 `index.html`
3. 替換圖片與文字內容為您的品牌資訊
