# Kinetic Studio Template

創意工作室品牌形象網站樣板，前衛 Syne 字體搭配橙黃活力配色，多層次滾動視差。

## 特色

- **滾動視差**：多層次背景元素不同速度移動，深度感強
- **Kinetic Typography**：大標題 highlight 效果，視覺衝擊力
- **活力配色**：暖白底 + 橙色主調 + 亮黃點綴
- **Syne 前衛字體**：幾何感強烈，極具辨識度
- **響應式設計**：完整適配桌面、平板、手機
