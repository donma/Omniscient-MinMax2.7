Python 資料科學與機器學習完整攻略

從零基礎開始，透過 Python 學習資料科學的核心概念，包括資料處理、視覺化、機器學習演算法，到實際專案實作。課程採用專案導向學習，讓你結訓時擁有完整的作品集。

40 小時
120 堂影片
終身存取
完課證書

你將學到

  • Python 基礎與進階語法
  • NumPy 與 Pandas 資料處理
  • Matplotlib 資料視覺化
  • Scikit-learn 機器學習
  • 監督式與非監督式學習
  • 深度學習基礎概念
