# 心跳基金會 Heartbeat Foundation

## 關於我們

心跳基金會成立於 2010 年，是一個致力於幫助弱勢兒童與家庭的非營利組織。我們相信每一個生命都值得被珍視與守護，因此長期投入教育支援、家庭扶助、健康關懷等公益服務。

## 我們的使命

用愛與行動，為弱勢族群創造更好的生活環境與未來。我們透過多元化的服務計畫，幫助需要幫助的人建立自信、獲得資源、並發揮潛能。

## 服務項目

- **教育支援計畫**: 獎助學金、課後輔導、學習資源
- **家庭扶助計畫**: 緊急生活扶助、就業輔導、庇護安置
- **健康關懷計畫**: 免費健康檢查、醫療補助、心理健康服務
- **社區發展計畫**: 社區培力、在地創生、社會企業支持
- **急難救助計畫**: 天然災害緊急救援、物資援助
- **國際人道援助**: 海外弱勢兒童援助、國際志願服務

## 影響力數據

十年來，我們已服務超過 50,000 個案，與 120 個合作機構攜手，培訓 8,000 人次志工，在 25 個城市留下愛的足跡。

## 加入我們

您可以透過以下方式支持我們：

- **捐贈善款**: 資助弱勢兒童與家庭的各項計畫
- **擔任志工**: 加入志願服務隊，用行動傳遞溫暖
- **企業合作**: 與企業夥伴共同推動社會公益
- **物資捐贈**: 捐贈圖書、文具、生活物資等
