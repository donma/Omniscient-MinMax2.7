# 沐光美學｜生成式 AI Prompt

## 樣板一句話定位
以 Split Layout 為核心的中高價位美容工作室網站，透過大圖＋文字的黃金比例營造專業信任感，適合強調服務品質與專業技術的美容 SPA 品牌。

## 適合產業
- 美容美學工作室 / SPA
- 個人美容職人品牌
- 肌膚管理中心
- 孕婦美容 / 產後護理
- 醫美診所（非的手術型）

## 不適合產業
- 電商購物網站
- 餐飲店面
- 科技 SaaS 產品
- 活動報名網站

## 視覺方向
暖土陶窯色系（背景 `#fef7f0`、主色 `#e07b53`、重點 `#2d5016`），傳遞溫暖、有機、值得信賴的品牌調性。字體選用 Outfit（標題）＋ Noto Sans TC（內文），呈現現代簡約的專業感。

## 色彩策略
- 背景：暖米色 `#fef7f0`
- 主色：赤陶橙 `#e07b53`（用在 CTA、按鈕、強調數字）
- 重點色：森林綠 `#2d5016`（用在標籤、信任數字、Footer）
- 文字：深褐色 `#1a1209`、副文字 `#6b5a4a`
- 間距：統一的 24px 基準、16px 卡片圓角

## 版型策略
本樣板採用「左右分割 Split Layout」為主軸：
- 首頁 Hero：左側大圖（充滿整個左半部）＋ 右側品牌主張
- 內頁使用一致的分段結構（圖文交替、左右對調）
- 服務區塊：三欄卡片（桌面）、單欄（手機）
- 流程區塊：四步驟橫向排列，帶連接線

## Hero Pattern
`split-image-text`：左邊 50% 區塊放滿版圖片（人物或環境照），右邊 50% 放品牌標題、敘述與 CTA。這個結構適合「形象很重要」的服務型品牌。

## Navigation Pattern
`top-navigation-mobile-drawer`：桌面版頂部橫向導覽，手機版漢堡選單右側。Header 固定且有毛玻璃效果（backdrop-filter: blur）。

## Content Rhythm
`service-trust-case-faq`：首頁從上到下依序是 Hero → 服務列表 → 品牌信任數據 → 作品集 → 顧客好評 → FAQ → CTA。

## 圖片風格要求
- Hero 圖：人物護理情境（女性臉部特寫、SPA 環境、空罐保養品）
- 服務圖：每項服務一張，建議 3:4 或 1:1 比例
- 作品集：4:3 比例，可做 hover overlay 顯示文字
- 顧客好評：可用真實顧客頭像，或留空（強調內容）
- 整體色調偏暖黃，不要過度銳利

## 圖片數量要求
- 首頁 Hero：1 張（1200×800 以上）
- 服務卡片圖：6 張（每項服務一張，建議 400×400）
- 作品集牆：6-8 張
- 團隊照：3 張（400×400）
- Blog 列表圖：6 張（800×450）
- 環境照：2-3 張
- 總計：約 25-30 張

## 12 頁內容定位
| 頁面 | 定位 |
|------|------|
| index.html | 首頁：完整呈現品牌實力（Hero + 服務 + 信任 + 作品 + 好評 + FAQ + CTA） |
| about.html | 品牌故事：創辦人背景、理念、團隊成員 |
| services.html | 服務總覽：6 項服務的詳細說明與價格區間 |
| service-detail.html | 單一服務深入介紹：適用對象、課程內容、預期效果 |
| portfolio.html | 作品案例牆：before/after 或情境照展示 |
| reviews.html | 顧客好評：精選 6 則真實回饋 |
| faq.html | 常見問題：8-10 個常見 Q&A |
| booking.html | 預約表單＋須知：表單 + 服務時間 + 取消政策 |
| process.html | 流程介紹：四步驟視覺化呈現 |
| blog.html | 部落格列表：6 篇文章，文章列表格式 |
| blog-detail.html | 文章內頁：完整文章內容，含標題、內文、圖片 |
| contact.html | 聯絡我們：地址地圖＋聯絡表單 |

## SEO 關鍵字策略
- 目標關鍵字：台北美容工作室、臉部保養推薦、敏感性肌膚護理、台北 SPA
- Title：網站名＋一句話描述
- Description：品牌定位＋服務項目＋地點
- 內頁各自獨立 meta，避免重複內容

## CTA 策略
- 主要 CTA：「立即預約」（出現在 Hero、服務區、CTA Section）
- 次要 CTA：「查看服務」、「看更多案例」
- 預約按鈕在 Header 固定顯示

## RWD 規則
- 桌機：雙欄、三欄、Grid 並存
- 平板：雙欄（服務、作品集）
- 手機：全部單欄，Hero 變成上下堆疊（圖在上、文在下）
- 手機漢堡選單：全屏覆蓋，中文導航

## 可替換內容
- 品牌名稱「沐光美學」
- 聯絡資訊（地址、電話、LINE）
- Google Map embed 連結
- 定價與服務時間
- 社群連結（LINE、IG）
- 所有文案與圖片

## 禁止事項
- 不可破壞整體色系的一致性（橙＋綠暖土色系）
- 不可將 Split Hero 改為全螢幕沉浸式（失去該版型特色）
- 不可刪除 Header 的預約 CTA 按鈕
- 不可使用過多黑色（壓迫感破壞美容產業調性）

## 與相似樣板的差異說明
本樣板不是一般「Hero + 三卡片 + FAQ + CTA」的公版結構：
- Hero 是左右分割而非上下排列，視覺記憶點更強
- 服務區用 6 欄網格而非 3 欄，信息量更豐富
- 信任區用深色背景對比白色數字，視覺層次更立體
- 所有內頁都呼應 Split Layout 的圖文對話結構
