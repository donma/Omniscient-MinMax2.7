# 野味餐酒｜生成式 AI Prompt

## 樣板一句話定位
適合 Fine Dining 餐酒館的 Magazine Editorial 風格網站，以大地色系、編輯式排版與故事敘事呈現餐飲美學與品牌深度。

## 適合產業
- Fine Dining 餐酒館
- 當代料理餐廳
- 預約制私廚
- 咖啡與甜點專門店
- 調酒吧

## 不適合產業
- 速食連鎖店
- 外送平台
- 超市/食品電商
- 活動報名

## 視覺方向
大地色系（背景 `#f5f0eb`、主色 `#8b4513`、重點 `#4a7c59`），搭配 Playfair Display 標題字體，創造 Editorial 雜誌質感。整體調性：有文化底蘊、不喧嘩、有故事。

## 版型策略
本樣板採用「Magazine Editorial」版型：
- 首頁 Hero：滿版大圖覆蓋左半邊，右側品牌主張（不同於 Template 001 的 Split）
- 圖文交替使用非對稱網格（Asymmetric Grid）
- 大量使用 large quotation、editorial 引言區
- 頁面間距較大，給文字呼吸空間

## Hero Pattern
`magazine-cover-full-bleed`：圖片幾乎覆蓋整個左半部（60-70%），右側內容區乾淨，留白充足。

## Navigation Pattern
`top-navigation-transparent`：Header 初始透明，滑動後變為毛玻璃背景，字體全部大寫、間距寬鬆。

## Content Rhythm
`editorial-asymmetric-storytelling`：首頁從上到下：Hero → 品牌哲學 → 非對稱菜單預覽 → 引言牆 → 故事章節 → 地圖預約 → Blog 預覽 → CTA。

## 圖片風格要求
- 餐飲相關：近拍食物（不一定整齊」，帶有溫度感）
- 環境照：餐桌、燭光、食材陳列
- 色調偏暖黃與土棕色
- 不過度修圖，保持食物真實感

## 圖片數量要求
- 首頁 Hero：1 張（1200×800 以上）
- 菜單圖：6-8 張（每道菜一張，800×600）
- 環境/場域圖：6 張
- 團隊/主廚圖：3 張
- Blog 圖：6 張
- 總計：約 25 張

## 12 頁內容定位
| 頁面 | 定位 |
|------|------|
| index.html | 首頁（編輯式呈現品牌哲學、菜單預覽、故事精華） |
| about.html | 品牌故事、主廚團隊 |
| services.html | 完整菜單（八道菜的套餐結構） |
| service-detail.html | 單一菜品深入介紹 |
| portfolio.html | 場域空間展示 |
| reviews.html | 顧客好評（編輯式列表） |
| faq.html | 常見問題（ FAQ 形式） |
| booking.html | 預約系統與須知 |
| process.html | 用餐流程（四個章節） |
| blog.html | 專欄文章列表 |
| blog-detail.html | 文章內頁（長文閱讀） |
| contact.html | 聯絡資訊＋地圖 |

## SEO 關鍵字
目標：台北 Fine Dining、預約制餐廳、在地食材餐廳、台北餐酒館

## CTA 策略
- 主要 CTA：「預訂座位」（明顯且隨處可見）
- 選單有「線上預約」按鈕

## RWD 規則
- 桌機：編輯式非對稱網格
- 平板：網格簡化為單欄
- 手機：全部單欄，Hero 改為上下堆疊

## 可替換內容
- 品牌名稱與 Slogan
- 菜單內容與價格
- 地址電話地圖
- 所有圖片

## 禁止事項
- 不要使用太現代的字體（如 Inter, Roboto）
- 不要破壞大地色系（棕色＋綠色）
- 不要將 Magazine 樣式改為公版 Hero 樣式
- 不要刪除「預訂座位」CTA

## 與 Template 001 的差異
Template 001 是 Split Layout + 信任感導向；Template 002 是 Magazine Editorial + 故事敘事導向。前者適合美容 SPA，後者適合有文化深度的餐飲品牌。
