# NEON MATRIX Template

## Overview
NEON MATRIX is a cyberpunk-themed gaming portal template featuring a dark aesthetic with vibrant neon accents.

## Pages (12)
1. index.html - Homepage
2. games.html - Game library
3. game-detail.html - Game details
4. news.html - News listing
5. news-detail.html - News article
6. esports.html - Esports section
7. community.html - Community forum
8. merch.html - Merchandise store
9. about.html - About us
10. contact.html - Contact form
11. faq.html - FAQ
12. careers.html - Job listings

## CSS Classes (Unique)
All classes use `-neon` suffix:
- Header: `.header-neon`, `.logo-neon`, `.nav-neon`, `.nav-link-neon`
- Buttons: `.btn-neon-green`, `.btn-neon-pink`
- Cards: `.game-card-neon`, `.news-card-neon`, `.team-card-neon`, `.merch-card-neon`, `.job-card-neon`, `.streamer-card-neon`
- Layout: `.hero-neon`, `.section-dark-neon`, `.section-alt-dark-neon`, `.grid-3-neon`, `.grid-4-neon`
- Effects: `.glitch-text-neon`, `.glow-line-neon`, `.particle-neon`
- Forms: `.form-neon`, `.form-input-neon`, `.form-textarea-neon`, `.form-select-neon`, `.form-label-neon`
- FAQ: `.faq-item-neon`, `.faq-question-neon`, `.faq-answer-neon`
- Footer: `.footer-neon`, `.footer-brand-neon`, `.footer-links-neon`

## JavaScript Features
- Navigation toggle for mobile
- Smooth scrolling
- FAQ accordion

- Scroll-triggered animations
- Form validation
- Counter animations
