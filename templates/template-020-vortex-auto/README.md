# VORTEX AUTO 汽車產業網站

## 主題說明
超級跑車與高端汽車經銷商網站，以深邃的暗色系呈現速度感與科技感。網站設計融合賽道美學與奢侈品網頁設計。

## 設計特色
- **色彩**: 深黑 #0A0A0F / 速度紅 #E63946 / 碳灰 #1A1A2E / 金屬銀 #C0C0C0
- **字體**: Orbitron（賽道科技感）+ Montserrat
- **首頁**: 全屏 Hero 疊加視差，車款展示幻燈片
- **特效**: 速度線裝飾、漸層疊加、hover 變形
